A simple instagram like application using Ruby On Rails. 



# Gems Used

  - boostrap-sass
  - devise
  - paperclip

# Installation
Make a project folder then open it:
```sh
$ mkdir photoboard && cd photoboard
```

Install dependencies:
```sh
$ bundle install
```

Migrate Database:
```sh
$ rake db:migrate
```

Run the server:
```sh
$ rails s
```
